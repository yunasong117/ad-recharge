☁ 클라우드 동기화 설정

Supabase를 연결하면 팀원들과 데이터를 실시간 공유할 수 있습니다. 한 명이 데이터를 변경하면 모든 PC에서 즉시 반영됩니다.

1단계 · Supabase 프로젝트 만들기

  1. supabase.com 접속 후 무료 가입 (GitHub 로그인 가능)
  2. "New Project" 클릭 → 이름/비밀번호 입력 → 생성 (1~2분 소요)
  3. 왼쪽 메뉴 SQL Editor 클릭
  4. 아래 SQL을 복사해서 붙여넣고 RUN 클릭:
create table ad_state (
  id text primary key,
  payload jsonb not null,
  updated_at timestamptz default now()
);

alter table ad_state enable row level security;

create policy "open access" on ad_state
  for all using (true) with check (true);

-- 실시간 동기화 활성화
alter publication supabase_realtime add table ad_state;

분류 규칙. 네이버파이낸셜·네이버페이 비즈월렛 → 스마트스토어 / 카카오_이니시스 → 카카오모먼트 / 카카오(정기결제)_다날 → 키워드광고 / 유니드컴즈·샐러드랩·킵그로우·컷백·채널코퍼레이션·캔바 → 정기결제 / 그 외 → 기타(원본 매체명 표시).

분석 방법. 일별 집계 → 평균/표준편차/요일별/주간 최대 계산 → 예측 모드(P90 안전 / P75 보수 / avg 균형 / P50 효율)로 회당 이체액 산정. 회당 이체액 = 일 기준값 × (5 영업일 ÷ 주당 빈도) × 브랜드 가중치로 계산하며, 백만원 단위로 내림 처리됩니다. 한도 초과 시 자동 캡 적용. 브랜드데이는 평소 "없음"으로 두고, 실제 브랜드데이가 있는 달에만 +30~50%로 한 단계 올리세요. 현재 잔액은 충전이나 사용이 있을 때마다 수동으로 업데이트해주세요.
